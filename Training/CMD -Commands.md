## File & Folder Commands

dir	                      =   Lists files and folders
cd	                      =   Changes directory
cd ..	                  =   Goes up one folder
mkdir foldername	      =  Creates a new folder
rmdir foldername	      =   Deletes a folder
del filename	          =  Deletes a file
copy source destination	  = Copies files
move source destination	  = Moves files
ren oldname newname       = Renames files
type filename	          = Shows file contents
